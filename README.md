# 🚀 Project Hub v2.1

A modern platform developed for **Vignan's Institute of Information Technology (VIIT)** to showcase student projects across different departments and academic years.

The platform enables students, faculty, and visitors to browse innovative projects through a clean, fast, and responsive interface.

---

## 🌐 Live Demo

🔗 https://project-hub-v2.netlify.app/
---

## ✨ Features

- 📂 Browse projects by Department and Academic Year
- 🔍 Fast project retrieval using Firebase Firestore
- 🖼️ Project image previews
- 📖 Detailed project information modal
- ⭐ Review & Rating system
- 📝 Submit new projects using Tally Forms
- 📱 Responsive UI for desktop and mobile
- ⚡ Faster loading with Firebase backend
- 🎨 Modern redesigned interface

---

- Cleaner interface
- Modern appearance
- Improved responsiveness
- Better user experience

### ⚡ Performance Improvements
- Faster project loading
- Efficient Firestore queries
- Improved project retrieval

### 📝 Tally Forms Integration
- Replaced Google Forms with Tally Forms
- Cleaner submission experience
- Easier project contribution workflow
